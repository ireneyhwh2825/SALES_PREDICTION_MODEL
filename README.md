# SALES_PREDICTION_MODEL
Machine Learning project that predicts BigMart product sales using Linear Regression and Random Forest Regression. Includes feature engineering, model evaluation, hyperparameter tuning, and business insights.

This project builds a machine learning model to predict product sales using the BigMart Sales dataset. The project follows the complete machine learning pipeline, including data exploration, feature engineering, data preprocessing, model training, evaluation, hyperparameter tuning, and business interpretation.

Two regression models were trained and compared to identify the model that provides the most accurate sales predictions.


##  Objectives

- Explore the sales dataset.
- Perform feature engineering.
- Handle missing values.
- Encode categorical variables.
- Split the dataset into training and testing sets.
- Train multiple regression models.
- Evaluate model performance using regression metrics.
- Visualize feature importance.
- Compare actual and predicted sales.
- Optimize the best-performing model using GridSearchCV.



##  Technologies Used

- Python 3
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn


## Machine Learning Pipeline

The project follows these steps:

1. Data Exploration
2. Feature Engineering
3. Data Cleaning
4. One-Hot Encoding
5. Feature Selection
6. Train-Test Split (80:20)
7. Linear Regression Model
8. Random Forest Regressor
9. Model Evaluation
10. Feature Importance Analysis
11. Actual vs Predicted Visualization
12. Hyperparameter Tuning using GridSearchCV
13. Business Interpretation



##  Models Used

- Linear Regression
- Random Forest Regressor




##  Business Value

The predictive model can help retailers:

- Forecast product sales
- Improve inventory planning
- Optimize pricing strategies
- Support data-driven business decisions


##  Conclusion

A complete machine learning workflow was implemented to predict product sales using the BigMart dataset. After comparing multiple regression models, the Random Forest Regressor achieved better predictive performance than the Linear Regression model. Hyperparameter tuning further improved the model, making it suitable for sales forecasting and business decision-making.
